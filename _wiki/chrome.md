## 快捷键


| 功能                      | 快捷键                                  |
|:--------------------------|:--------------------------------------|
| 重新打刚关系的网页     | shift+cmd+t                      |
| 新打开浏览器 | cmd+n                            |
| 强制刷新                     | shift+cmd +r                     |
| 左右切换页面    | option+cmd+左右箭头键                |
| 新打开标签            | cmd+t                |
| 打开历史记录           | cmd+y               |
| 打开清除浏览数据           | shift+cmd+delete              |
| 地址栏光标移动到开头或结尾           | cmd+左右箭头键                |
